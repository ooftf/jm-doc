# 客户端管理

## 1. 判断是否在客户端内

```js
function isInJm() {
  var ua = navigator.userAgent;
  return /JM_IOS|JM_ANDROID/i.test(ua);
},
```

## 2. 获取客户端版本

```js
function getJmVersion () {
  var ua = navigator.userAgent;
  var match = ua.match(/(JM_IOS|JM_ANDROID)\/([\d\.]+)/i)
  if (match) {
    return match[2]
  }
  return 0
}
```
